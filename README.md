## 1. Installation

You can install `komau` with your plugin manager:

```vim
Plug 'ntk148v/komau.vim' " Vim-plug
```

```lua
use {"ntk148v/komau.vim"} -- Packer
```

## 2. Usage

```vim
:colorscheme komau
# For light version
:set background=light
# For dark version
:set background=dark
```

## 3. Credits
